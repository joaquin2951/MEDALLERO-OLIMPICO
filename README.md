<h1> TRABAJO PRÁCTICO GRUPAL REALIZADO PARA UTN FRSF </h1>
<h2> Version final en: parte3/MedalleroOlimpico </h2>
<h2> OBJETIVOS: </h2>
<p> En el trabajo práctico integrador, se desarrolló una aplicación, que 
permita a los usuarios registrados acceder a gestionar un medallero olímpico. Esta aplicación 
tiene como objetivo principal registrar y mostrar las medallas obtenidas por los países 
participantes en diversas disciplinas deportivas. 
En primer lugar, se especificó la estructura de menús de la interfaz de usuario, 
definiendo claramente las opciones para registrar medallas, consultar el medallero y generar 
reportes. Luego, se pasó a definir y codificar los algoritmos necesarios para la gestión 
eficiente de los datos del medallero. </p> 
<h3> Conocimientos adquiridos: </h3>
<p> • Estrategia de descomposición modular descendente.  <br>
• Funciones y recursividad.  <br>
• Tipos de datos abstractos.  <br>
• Archivos de textos y directos para el almacenamiento persistente de los datos del 
medallero. </p>
